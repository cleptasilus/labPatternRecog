\section*{Task 1.1}

\subsection*{Problem}
Why do we need fix for Matrix A and round for Matrix B?

\subsection*{Solution}
Because for A, we want to round down as we use the interval $[0,10)$ and for B we don't exclude 10 and therefore need to round correctly.
